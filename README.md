# ❄️ Frost-Gaming Module V1

A powerful Magisk Module designed to enhance gaming performance, stabilize FPS, and unlock high-graphics settings for Infinix and other Android devices.

## 🚀 Features
- **Device Spoofing:** Spoofs your device as **Pixel 8 Pro** to unlock high-graphics options and exclusive features in games like MLBB, PUBG, and more.
- **Advanced GPU Rendering:** Forces **Skia GL** rendering for smoother visuals and reduced CPU overhead.
- **Extreme Thermal Bypass:** Disables thermal throttling (stops thermal-engine/thermald) to prevent FPS drops during intense gameplay.
- **System Optimizations:** Disables unnecessary logging and optimizes network buffers for a more stable ping.
- **Touch Responsiveness:** (Beta) Improved touch sampling for faster reaction times.

## ⚠️ Warning
- **Heat Management:** Disabling thermal throttling will make your device run hotter. It is **highly recommended** to use a phone cooler/fan while gaming for long periods.
- **Risk:** Use this module at your own risk. I am not responsible for any damage to your hardware.

## 🛠️ How to Install
1. Download the `Frost-Gaming-V1.zip` from the [Releases](link-to-your-release-here) section.
2. Open **Magisk** or **Kitsune Mask**.
3. Go to the **Modules** tab.
4. Click **Install from storage** and select the ZIP file.
5. Wait for the process to finish and **Reboot** your device.

## 📱 Compatibility
- Tested on **Infinix** series.
- Universal: Works on **all Android** devices
- Requires **Magisk 20.4+**.
